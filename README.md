# Code Sportif FMMAF

Guide de révision interactif du **Code Sportif FMMAF v9.0** (Janvier 2025) — règles officielles des compétitions de MMA Jeunes & Amateurs de la Fédération Française de MMA.

## Contenu

- Définition & durée des combats
- Les officiels (arbitre, juges, médecin…)
- Le jugement (système 10 points, critères 3D)
- Coups interdits et procédures
- Catégories de poids (U14, U16, U18, U21, 21+)
- Règles évolutives par âge
- Équipements obligatoires
- Types de résultats (soumission, KO/TKO, décisions…)
- Dispositif médical

## Stack

- [Astro](https://astro.build) — générateur de site statique
- Déploiement automatique sur GitHub Pages via GitHub Actions

## Développement

```bash
npm install
npm run dev
```
